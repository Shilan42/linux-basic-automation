## Bash Script для Практического Задания

### Описание
Данный репозиторий содержит bash-скрипт для выполнения практического задания по работе с файловой системой и базовыми командами Linux.

### Функционал скрипта
Скрипт выполняет следующие операции:

* Создание структуры каталогов
* Работа с файлами и правами доступа
* Манипуляции с содержимым файлов
* Использование переменных окружения
* Поиск и обработка текста
* Управление файловой системой

### Структура каталогов
При выполнении скрипта создается следующая структура:

```
task/
├── dir1/
│   └── summary.txt
├── dir2/
│   ├── empty
│   ├── hello.sh
│   └── list.txt
└── dir3/
    └── dir4/
        ├── empty
        ├── hello.sh
        └── list.txt
```

### Основные возможности
* Создание каталогов и файлов
* Настройка прав доступа
* Копирование и перемещение файлов
* Работа с потоками ввода-вывода
* Использование переменных окружения
* Поиск и обработка текста в файлах

### Инструкция по использованию
1. Клонируйте репозиторий:
```bash
git clone https://github.com/your-username/your-repo.git
```

2. Перейдите в директорию проекта:
```bash
cd your-repo
```

3. Сделайте скрипт исполняемым:
```bash
chmod +x script.sh
```

4. Запустите скрипт:
```bash
./script.sh
```

### Зависимости
* Bash (версия 4.0 или выше)
* Стандартные утилиты Linux (mkdir, touch, chmod, cp, find, grep)

### Примечания
* Скрипт автоматически удаляет созданную директорию task после выполнения
* Все операции выполняются в текущей директории
* Результаты работы сохраняются в файле "Практическое задание.txt"
